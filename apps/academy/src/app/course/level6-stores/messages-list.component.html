METODOLOGÍA CENTRALIZADA

Para aplicaciones a gran escala, mantener el estado distribuido en componentes produce incoherencias. Con @ngrx/signals, unificamos el estado en un store autónomo mediante signalStore(). Esto permite derivar eficientemente computadas reactivas con withComputed() y mutar de forma segura el estado inmutable usando withMethods() y patchState().
