CENTRALIZED METHODOLOGY

For large-scale applications, keeping state distributed across components causes inconsistencies. With @ngrx/signals, we unify state in an autonomous store using signalStore(). This allows us to efficiently derive reactive computed properties with withComputed() and safely mutate immutable state using withMethods() and patchState().
